# springboot-rest-api-demo
Spring Boot REST API demo with JPA, Hibernate and Swagger
This project demonstrates a simple REST API built using Spring Boot, Spring Data JPA, and Hibernate.

## Features
- CRUD REST APIs
- Spring Boot + Hibernate
- JPA Entity mapping
- Global exception handling
- Swagger (OpenAPI) documentation
- H2 in-memory database

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
- H2 Database

## How to Run
- mvn spring-boot:run
- Access Swagger UI: /swagger-ui.html
